# Бот для конвертации валют
Этот бот разработан для конвертации валют с использованием Telegram Bot API и библиотеки currency_converter.

## Установка
Установите необходимые зависимости, выполнив следующую команду:

```p
pip install pyTelegramBotAPI currency_converter
```
Получите токен бота от BotFather в Telegram.

Запустите бота, используя скрипт `bot.py`.

## Использование
Начните диалог с ботом, написав команду `/start`.

Введите сумму, которую нужно конвертировать.

Выберите пару валют для конвертации из предложенных вариантов или введите свою пару значений.

Получите результат конвертации.
